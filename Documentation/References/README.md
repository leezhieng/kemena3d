# API Reference

The API reference is generated from source code comments using Doxygen and is not committed to the repository.

## Generating the Reference

From the project root, run:

```bash
python build_documentation.py
```

The output will be available at `Documentation/References/html/index.html`.

## Requirements

- [Python](https://www.python.org/downloads/) 3.6 or later
- [Doxygen](https://www.doxygen.nl/download.html) 1.9 or later
- _(Optional)_ [Graphviz](https://graphviz.org/download/) — enables class diagrams and call graphs
